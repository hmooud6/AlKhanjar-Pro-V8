## البناء والتثبيت

### 1. بناء التطبيق:

```bash
cd android-app
./gradlew assembleRelease
```

الملف الناتج: `app/build/outputs/apk/release/app-release.apk`

### 2. تشغيل لوحة التحكم:

```bash
cd web-dashboard
# افتح index.html في المتصفح
# أو استخدم خادم ويب
python3 -m http.server 8000
```

## إعداد Firebase:

1. انسخ `google-services.json` إلى `android-app/app/`
2. تأكد من قواعد Firebase:

```json
{
  "rules": {
    ".read": true,
    ".write": true
  }
}
```
